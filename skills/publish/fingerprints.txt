# fingerprints.txt — dev953 de-attribution pattern list.
#
# This is DATA, NOT a scanner. It carries no logic and runs nothing. The publish
# skill (skills/publish/SKILL.md) reads these patterns and applies a MECHANICAL
# regex/string substitution across the throwaway temp clone, producing a diff —
# never a generative rewrite. The secret/PII scan logic lives ONCE in the hook
# (hooks/scan.sh); this file is unrelated to that and never duplicates it.
#
# FORMAT: one rule per non-comment, non-blank line.
#   <case-insensitive extended-regex>[<TAB><replacement>]
# The pattern runs to the first TAB (or to end-of-line if there is no TAB). A
# bare pattern with NO TAB means "delete the matched text". An optional TAB
# introduces a literal replacement string when one is wanted instead of deletion.
# Every rule here is delete-only, so none carries a TAB. Whole lines that become
# empty after substitution (e.g. a co-author trailer) are dropped by the skill.
# Patterns are applied to file contents, comments, README/docs, package metadata,
# dotfiles/CI, and .git author/committer fields. Lines beginning with '#' and
# blank lines are ignored.
